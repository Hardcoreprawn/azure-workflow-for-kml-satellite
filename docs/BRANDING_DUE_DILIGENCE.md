| **French** | ✅ Clean. "Canopée" is French for "canopy" — the prefix evokes the same positive association. No offensive reading. |
| **German** | ✅ Clean. No German word contains "canop-". "Kanopee" (canopy) is a rare loanword but positive. |
| **Spanish** | ✅ Clean. No Spanish word or slang. "Canopia" (canopy) is a recognisable root. |
| **Portuguese** | ✅ Clean. "Canópia" / "copa" are canopy-related. No negative association. |
| **Italian** | ✅ Clean. No issues. |
| **Mandarin** | ✅ Phonetically "ka-no-peks" — no overlap with offensive Mandarin syllables. |
| **Japanese** | ✅ Katakana-friendly: カノペックス. No issue. |
| **Arabic** | ✅ No phonetic overlap with offensive terms. |
| **Pharmaceutical** | ✅ **No drug called "Canopex" exists.** Drugs.com returns zero results. The "-ex" suffix is common in pharma (Xanax, Prozac patterns use different suffixes), but Drugs.com's closest suggestion was "Casodex" (a cancer drug) — phonetically distant. No confusion risk. |
| **Urban slang** | ✅ No Urban Dictionary entry. No meme, no sexual slang, no derogatory meaning. |

**Overall linguistic risk: VERY LOW.** Universally clean across all major languages in the target market (EU, UK, Americas, Asia-Pacific).

#### Arboryx

| Language / Region | Assessment |

| **Urban slang** | ✅ No Urban Dictionary entry. No meme association. |

**Overall linguistic risk: LOW.** One minor Tagalog slang note, but not material for the target market.

| **Namespace crowding** | ✅ "Canop-" prefix is uncrowded. Only "Canopy" in the adjacent space, and we're clearly distinct. | ⚠️ "Arbor-" prefix is moderately crowded: Arboris (rejected), Arbora (rejected), Arboreal, Arboretum. More brand-adjacent noise to cut through. |

| Namespace distinctiveness | **Canopex** | Moderate — "canop-" is less crowded than "arbor-" |

### 5.6 Recommendation

**Canopex wins on every friction dimension and ties on everything else.**

The deciding factor isn't any single issue — it's the cumulative effect of
pronunciation clarity + spelling resilience + word-of-mouth transmission.
Arboryx is a perfectly good name that would work fine. But Canopex is
*frictionless* in ways that matter for a self-serve SaaS product where
word-of-mouth and organic search are the primary growth channels.

The semantic bonus is real too: "canopy" implies an overhead perspective on
forests, which is literally what the product does (satellite imagery of forest
canopy). "Arbor" implies trees in general, which is correct but less evocative
of the satellite angle.

**Recommendation: proceed with Canopex.**

Arboryx remains an excellent fallback if any trademark attorney uncovers a
blocking issue with the TMView partial matches for Canopex.
